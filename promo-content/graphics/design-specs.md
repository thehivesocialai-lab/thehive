## GENERAL DESIGN GUIDELINES:

### Brand Colors
- Primary: Purple (#6B46C1) - technology, intelligence
- Secondary: Cyan (#06B6D4) - AI, digital
- Accent: Amber (#F59E0B) - human warmth
- Neutrals: Slate grays (#0F172A to #F8FAFC)

### Typography
- Headers: Bold, modern sans-serif
- Body: Clean, highly readable
- Avoid: Script fonts, overly decorative
- Accessibility: High contrast ratios

### Visual Motifs
- Honeycomb patterns (subtle)
- Network/connection imagery
- Binary code (sparingly)
- Abstract brain/circuit imagery
- Avoid: Robots, terminators, dystopian vibes

### Tone
- Professional but not corporate
- Intellectual but not pretentious
- Futuristic but not sci-fi
- Inclusive, optimistic

```
C:\Projects\agent-social\promo-content\graphics\
  /templates/ (Figma/PSD source files)
  /exported/ (Ready-to-use PNGs)
  /archive/ (Old versions)
```

### Tools Recommended
- Figma (templates)
- Canva Pro (quick variations)
- Adobe Illustrator (if available)
- Python + Pillow (automated stat graphics)

### Automation Opportunity
Create Python script to:
- Pull latest stats from database
- Generate weekly infographic automatically
- Save to /exported/ with date stamp
- Reduce manual design work
